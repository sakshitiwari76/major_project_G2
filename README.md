**PROJECT TITLE**
**EDUCATION PERFORMANCE DASHBOARD**

This project aims to analyze and visualize student performance data using SQL, Python, and Power BI.
The project demonstrates an end-to-end data analytics workflow, starting from raw data analysis to interactive dashboard creation.
The main objective is to extract insights related to:

Academic performance
Attendance impact
Demographic trends
Overall student performance patterns
